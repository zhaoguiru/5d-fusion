# Five-Dimensional Synergy Framework for Magnetic Fusion

A unified five-dimensional diagnostic framework for magnetic-confinement fusion. Every fusion entity—whether a conceptual reactor design or an experimental plasma discharge—is treated as an ontological body characterized by five irreducible dimensions: **Boundary (B)**, **Structure (S)**, **Reserve (R)**, **Direction (D)**, and **Intensity (I)**. The internal synergy coefficient k_in quantifies the degree of mutual dimensional balance.

This repository supports the paper *"A Five-Dimensional Synergy Framework for Magnetic Fusion: From Reactor Design Screening to Plasma Discharge Diagnosis"* (submitted to *Plasma Physics and Controlled Fusion*).

**Repository contents:** `part1_reactor_design/` contains the 1,028-concept dataset (`fusion_5d_ready.csv`) and analysis scripts (`lhd_physical_bsrdi.py`, `lhd_analysis_advanced.py`). `part2_lhd_plasma/` contains the LHD stream parser (`lhd_5d_v6e.py`), statistical analysis (`analyze_lhd_5d.py`), physical validation (`validate_v6e_fixed.py`, `validate_v6e_summary.py`), and supplementary data generator (`generate_supplementary_data.py`). `paper/` holds the LaTeX manuscript.

**Prerequisites:** `pip install numpy pandas scipy matplotlib`

**Reactor design screening:** Run `lhd_physical_bsrdi.py` to detect B/S/R/D/I columns, normalize, and compute k_in; then `lhd_analysis_advanced.py` for ranking, Q-correlation, and quadrant figures. Key results: ARIES-AT (k_in=0.225), ARIES-I (0.108), CREST (0.105); Spearman rho with Q is 0.7729 (n=74); quadrant split Q1/Q2/Q3/Q4 = 39.6/10.4/39.6/10.4%.

**LHD plasma diagnosis:** Run `lhd_5d_v6e.py` to parse 9,712 raw LHD discharges (requires PRM+DAT files from [NIFS](https://www.lhd.nifs.ac.jp/)); then `analyze_lhd_5d.py` for 10 figures. Run `validate_v6e_fixed.py summary.csv output_dir` for cross-diagnostic consistency validation, Q3 t-test, and event correlation. Key results: mean B=0.932, S=0.072, R=0.942, D=0.002, I=0.525, k_in=0.115 +/- 0.042; r(B,R)=0.988; Q3 transients show highest synergy (k_in=0.167, t=65.31, p<<0.001); k_in is diagnostic-independent (p=0.12) and negatively correlated with event severity (r=-0.643). Run `generate_supplementary_data.py` to produce `Supplementary_Data_S1.csv` and `Supplementary_Data_S2.csv` for journal submission.

**Data:** `fusion_5d_ready.csv` (125 KB, 1,028 designs + 1,000 virtual scans) is included. LHD raw data (~45 GB) is not included; download from NIFS. LHD results `summary.csv` (1.44 MB) is generated by `lhd_5d_v6e.py`.
