Five-Dimensional Synergy Framework for Magnetic Fusion

A unified five-dimensional diagnostic framework for magnetic-confinement fusion. Every fusion entity—whether a conceptual reactor design or an experimental plasma discharge—is treated as an ontological body characterized by five irreducible dimensions: Boundary (B), Structure (S), Reserve (R), Direction (D), and Intensity (I). The internal synergy coefficient k_in quantifies the degree of mutual dimensional balance.

This repository supports the paper "A Five-Dimensional Synergy Framework for Magnetic Fusion: From Reactor Design Screening to Plasma Discharge Diagnosis" (submitted to Plasma Physics and Controlled Fusion).

Repository contents: part1_reactor_design contains the 1,028-concept dataset (fusion_5d_ready.csv) and analysis scripts (lhd_physical_bsrdi.py, lhd_analysis_advanced.py). part2_lhd_plasma contains the LHD stream parser (lhd_5d_v6e.py), statistical analysis (analyze_lhd_5d.py), physical validation (validate_v6e_fixed.py, validate_v6e_summary.py), P0 stored energy and confinement time validation (validate_physics_p0.py), P1 Q3 sensitivity analysis against S/D perturbations from -50% to +50% (q3_sensitivity_analysis2.py), and supplementary data generator (generate_supplementary_data.py). The paper directory holds the LaTeX manuscript. part2_lhd_plasma also contains summary.csv (1.44 MB), which is generated by lhd_5d_v6e.py.

Prerequisites: pip install numpy pandas scipy matplotlib

Reactor design screening workflow: run lhd_physical_bsrdi.py to detect B/S/R/D/I columns, normalize, and compute k_in; then run lhd_analysis_advanced.py for ranking, Q-correlation, and quadrant figures. Key results include ARIES-AT (k_in=0.225), ARIES-I (0.108), CREST (0.105); Spearman rho with Q is 0.7729 (n=74); quadrant split Q1/Q2/Q3/Q4 equals 39.6/10.4/39.6/10.4 percent.

LHD plasma diagnosis workflow: run lhd_5d_v6e.py to parse 9,712 raw LHD discharges (requires PRM+DAT files from NIFS at www.lhd.nifs.ac.jp); then run analyze_lhd_5d.py for 10 figures. Key results include mean B=0.932, S=0.072, R=0.942, D=0.002, I=0.525, k_in=0.115 plus or minus 0.042; correlation r(B,R)=0.988. Run validate_v6e_fixed.py with summary.csv and an output directory for cross-diagnostic consistency validation. Run validate_v6e_summary.py with summary.csv and an output directory for Q3 t-test and event correlation. Q3 transients show highest synergy (k_in=0.167, t=65.31, p much less than 0.001); k_in is diagnostic-independent (p=0.12) and negatively correlated with event severity (r=-0.643).

Physical validation and sensitivity workflow: run validate_physics_p0.py with --summary-csv pointing to summary.csv, --data-root pointing to the LHD extracted data path, and --output for the results directory. This validates k_in against estimated W_th and tau_E, outputting fig01 through fig06. Run q3_sensitivity_analysis2.py with summary.csv as the argument to test whether Q3 high synergy is robust against S and D perturbations from -50% to +50%.

Supplementary data generation: run generate_supplementary_data.py to produce Supplementary_Data_S1.csv (design screening) and Supplementary_Data_S2.csv (LHD discharge summary) for journal submission.

Data notes: fusion_5d_ready.csv (125 KB, 1,028 designs plus 1,000 virtual scans) is included. LHD raw data (~45 GB) is not included; download from NIFS. summary.csv (1.44 MB) is generated by lhd_5d_v6e.py.
