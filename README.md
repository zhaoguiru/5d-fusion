# Five-Dimensional Synergy Framework for Magnetic Fusion

A unified five-dimensional diagnostic framework for magnetic-confinement fusion, covering both **reactor design screening** (Part I) and **plasma discharge diagnosis** (Part II).

---

## Overview

This repository contains the code and data for the paper:

> **A Five-Dimensional Synergy Framework for Magnetic Fusion: From Reactor Design Screening to Plasma Discharge Diagnosis**

The framework treats every fusion entity—whether a conceptual reactor design or an experimental plasma discharge—as an ontological body characterized by five irreducible dimensions: **Boundary (B)**, **Structure (S)**, **Reserve (R)**, **Direction (D)**, and **Intensity (I)**. The internal synergy coefficient \(k_{in}\) quantifies the degree of mutual dimensional balance.

## Part I: Reactor Design Screening

### Dataset
`fusion_5d_ready.csv` contains 1,028 records spanning:
- **Existing/near-term devices**: LHD, ITER, SPARC, FIRE, K-DEMO
- **Conceptual designs**: ARIES-AT, ARIES-I, CREST, DEMO, CFETR, EU-DEMO, etc.
- **Virtual scans**: 1,000 parameter-sweep configurations

Columns: `device`, `shot_id`, `mode`, `R_m`, `tauE_s`, `Pfus_MW`, `Ip_MA`, `Bt_T`, `Q`

### Usage

```bash
cd part1_reactor_design

# Step 1: Detect B/S/R/D/I columns, normalize, compute k_in
python3 lhd_physical_bsrdi.py
# Output: lhd_5d_physical_bsrdi.csv

# Step 2: Ranking, Q-correlation, quadrant classification, figure generation
python3 lhd_analysis_advanced.py
# Outputs: fig23_kin_vs_Q.png, fig24_quadrant_colored.png, fig25_heatmap_gamma.png
```

### Key Results
- **Top coherence**: ARIES-AT (\(k_{in}=0.225\)), ARIES-I (0.108), CREST (0.105)
- **Q-correlation**: Spearman \(ho = 0.7729\) (\(n=74\))
- **Quadrant split**: Q1 Ideal 39.6% | Q2 Treasure 10.4% | Q3 Eliminate 39.6% | Q4 Trap 10.4%

## Part II: LHD Plasma Discharge Diagnosis

### Dataset
- **Source 1**: 59,810 PRM+DAT files → 1,167 multi-channel shots
- **Source 2**: 8,547 standalone concatenated DAT files
- **Total**: 9,714 shots, 9,712 successfully parsed (99.98%)

### Prerequisites
```bash
pip install numpy pandas scipy matplotlib
```

### Usage

```bash
cd part2_lhd_plasma

# Step 1: Parse LHD raw data and compute 5D + k_in for each shot
# (Requires lhd_raw/ directory with .prm and .dat files)
python3 lhd_5d_v6e.py
# Output: lhd_5d_results_v6e/summary.csv

# Step 2: Statistical analysis and figure generation
python3 analyze_lhd_5d.py
# Outputs: fig1_5d_distributions.png ~ fig10_clustering.png
```

### Key Results
- **Mean profile**: \(B=0.932\), \(S=0.072\), \(R=0.942\), \(D=0.002\), \(I=0.525\)
- **Mean \(k_{in}\)**: \(0.115 \pm 0.042\)
- **Strongest correlation**: \(r(B,R)=0.988\)
- **Counter-intuitive finding**: Q3 (low B, low R) exhibits highest \(k_{in}=0.167\)
